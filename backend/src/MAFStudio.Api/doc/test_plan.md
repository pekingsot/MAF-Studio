# 无水印视频解析工具测试计划

## 一、测试目标
确保视频解析工具在复杂网络环境下稳定运行，能够应对目标平台的反爬机制变化，保证广告激励系统公平有效，为用户提供可靠的服务体验。

---

## 二、测试范围

### 2.1 功能测试
- 视频链接识别与解析
- 无水印视频下载
- 广告激励系统
- 用户权限管理

### 2.2 非功能测试
- 反爬机制稳定性测试（重点）
- 性能与并发测试
- 安全与防作弊测试
- 兼容性测试
- 容灾与恢复测试

---

## 三、核心测试策略

### 3.1 反爬机制稳定性测试（优先级：P0）

#### 测试场景
| 场景编号 | 测试场景 | 预期结果 | 验证方法 |
|---------|---------|---------|---------|
| AP-001 | 平台签名算法变更 | 系统自动切换备用解析通道，≤5分钟恢复服务 | 监控告警+自动降级验证 |
| AP-002 | 平台封禁IP | 代理池自动轮换，用户无感知 | 模拟IP封禁场景 |
| AP-003 | 平台增加设备指纹校验 | WebView模拟方案生效，解析成功率>85% | 对比API直调与WebView方案 |
| AP-004 | 平台限频（如每小时解析次数限制） | 队列排队机制生效，提示预计等待时间 | 压测触发限频阈值 |
| AP-005 | 平台接口结构变化（字段重命名/删除） | 规则库热更新生效，无需重启服务 | 灰度发布规则变更 |
| AP-006 | 多平台并行解析压力 | 各平台独立降级，互不影响 | 同时触发抖音+快手解析失败 |
| AP-007 | 第三方解析服务故障 | 本地兜底方案启用或友好错误提示 | Mock第三方服务超时 |

#### 自动化监控指标
- 各平台解析成功率（按小时统计，低于90%触发告警）
- 解析平均耗时（P95 < 10秒）
- 反爬对抗失败次数（按平台/错误类型分类）

### 3.2 广告激励系统测试（优先级：P0）

#### 功能测试用例
| 用例ID | 测试步骤 | 预期结果 |
|--------|---------|---------|
| AD-001 | 首次使用触发广告 | 正确拉起微信激励视频广告 |
| AD-002 | 广告播放中途关闭 | 提示"需完整观看"，不授权当日使用 |
| AD-003 | 广告播放完成 | 后端记录用户状态，当日无限使用 |
| AD-004 | 同一用户当日第二次打开 | 无需再看广告，直接使用 |
| AD-005 | 跨日后首次使用 | 重新触发广告观看 |
| AD-006 | 广告加载失败 | 提示用户稍后重试，不消耗当日免费次数 |
| AD-007 | 弱网环境下广告播放 | 超时后友好提示，不卡死界面 |

#### 防作弊测试
| 用例ID | 作弊手段 | 预期结果 |
|--------|---------|---------|
| AC-001 | 修改本地时间绕过日期校验 | 后端时间戳校验，前端篡改无效 |
| AC-002 | 抓包重放广告完成回调 | 服务端幂等性校验，拒绝重复请求 |
| AC-003 | 模拟广告完成事件 | 微信官方SDK签名验证，伪造无效 |
| AC-004 | 多设备共享同一账号 | 后端设备ID绑定，限制并发使用 |

### 3.3 视频解析核心功能测试

#### 平台覆盖矩阵
| 平台 | 链接格式 | 解析字段 | 优先级 |
|------|---------|---------|--------|
| 抖音 | 短链/长链 | 视频URL、标题、作者、封面 | P0 |
| 快手 | 分享链接 | 视频URL、标题、作者 | P0 |
| 小红书 | 笔记链接 | 视频/图片、标题、作者 | P1 |
| 西瓜视频 | 页面链接 | 视频URL、标题 | P1 |
| 微视 | 分享链接 | 视频URL、标题 | P2 |

#### 边界值测试
- 超长视频（>5分钟）
- 已删除/私密视频
- 直播回放链接
- 图文混合内容
- 含特殊字符的标题/作者名

### 3.4 性能测试

#### 并发测试指标
| 场景 | 并发数 | 目标响应时间 | 错误率阈值 |
|------|--------|------------|----------|
| 链接解析 | 100 | P95 < 8s | < 5% |
| 视频下载 | 50 | P95 < 15s | < 3% |
| 广告状态查询 | 200 | P95 < 500ms | < 1% |

#### 长期稳定性测试
- 连续运行72小时，监控内存泄漏
- 模拟每日10000次解析请求，观察服务降级情况

## 七、风险与应对

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|---------|
| 平台反爬升级导致解析失败 | 高 | 严重 | 建立多通道冗余、24小时监控 |
| 微信广告政策变更 | 中 | 中等 | 关注官方公告、保留降级方案 |
| 版权投诉导致服务下线 | 中 | 严重 | 用户协议免责条款、举报机制 |
| 高并发下服务崩溃 | 低 | 严重 | 压测验证、弹性扩容、熔断机制 |
